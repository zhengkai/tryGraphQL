nodejs 上的 hello world
---------------------

随便找个空目录

	git clone git@github.com:graphql/graphiql.git
	cd graphiql
	npm upgrade
	npm run dev

之后就可以访问本地 demo 了 `http://127.0.0.1:8080/`
